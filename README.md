# Global Sales Dashboard

## Project Overview
This dashboard provides insights into sales performance across different countries, products, and customer segments.

## Tools Used
- Power BI
- Microsoft Excel

## Dataset
Global sales Dataset

## KPIs
- Total Sales
- Total Profit
- Total Units Sold
- Average Discount

## Dashboard Features
- Sales by Country
- Sales by Product
- Sales by Segment
- Monthly Sales Trend
- Profit Analysis
- Top Performing Products

## Key Insights
- Identified top-performing countries and products
- Analyzed monthly sales trends
- Compared profit across segments
- Evaluated discount impact on sales performance
